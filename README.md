# 🏡 House Market Analysis Dashboard | Power BI

## 📌 Project Overview

This end-to-end Power BI project provides a comprehensive analysis of the housing market by examining sales performance, property characteristics, and house type trends. The dashboard helps stakeholders understand market behavior and supports informed business decisions using interactive visualizations.

---

## 🎯 Objectives

- Analyze housing market trends
- Monitor sales performance
- Compare different house types
- Identify key sales drivers
- Provide actionable business insights

---

## 📈 Dashboard Pages

### 1. House Market Overview
- Market overview
- Sales trends
- Property distribution
- Key performance indicators

### 2. Sales Performance
- Sales analysis
- Revenue comparison
- Category performance
- Key Drivers Analysis
- Detailed transaction table

### 3. House Type Analysis
- House type comparison
- Property distribution
- Trend analysis
- Interactive filtering using slicers
